# WaterColorFilter for Unity

[Unity Package](WaterColorFilter.unitypackage)

[![Thumbnail](Thumbnail600.jpg)](https://vimeo.com/152248671)

## Features
 - Tone-based Diffuse Shading
 - Wobbing
 - Edge Darkening
 - Paper Effect
