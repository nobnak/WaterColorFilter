## Features
 - Tone-based Diffuse Shading
 - Wobbing
 - Edge Darkening
 - Paper Effect

## Installation
Add scoped repository:

- URL: https://package.openupm.com
- Scope: jp.nobnak

Install "Water color filter" on Package Manager.
